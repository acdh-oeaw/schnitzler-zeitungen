Mithilfe

Gerne können Sie dazu beitragen, die automatische Volltexterkennung zu verbessern. Dies geschieht auf der Plattform Transkribus, für die eine einmalige kostenlose Anmeldung benötigt wird, die Sie hier vornehmen können: https://transkribus.eu

Für die Verbesserung von Texten kann auf die Website https://transkribus.eu/r/read/projects/43497 zugegriffen werden. Die Installation eines Programmes ist nicht nötig. Dieses kann aber alternativ verwendet werden.
